WebClassの資料ファイルをダウンロードできるブックマークレットです。

## 使い方
Releasesにあるファイルをダウンロードし、ブックマークレットとして登録しWebClassで使用できます。<br>
テキストブックページ (`txtbk_frame.php`) を開いて実行してください。ことあるごとに別ウィンドウや新しいタブで開いて`Ctrl + S`を押す手間を省くことができます。<br>
開かれているものがファイルでない場合 (表示領域の上部に`別ウィンドウで開く`ボタンが存在しない場合) 、実行できません。
同じページに複数のファイルが存在する場合、目次で見た時に上に位置するものから順にダウンロードされます。

ダウンロードしたファイルは、ブラウザでダウンロード用と指定されたディレクトリに保存されます。ブラウザのダウンロード履歴欄にも表示されます。<br>
複数ファイルを一度にダウンロードする場合、スパムと判定されてブラウザがブロックする可能性があります。そのようなときはサイトごとの設定で明示的に許可してください。
